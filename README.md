# 🏦 CreditWise Loan Approval System
AI-powered loan approval prediction system using Machine Learning and Streamlit.
An AI-powered Machine Learning web application that predicts whether a customer’s loan should be approved or rejected based on financial details.

## 🚀 Features

* Loan approval prediction
* Machine Learning classification model
* Interactive Streamlit web app
* User-friendly interface
* Data visualization

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Matplotlib

## 📊 Input Features

* Applicant Income
* Credit Score
* Existing Loans
* Savings
* Loan Amount
* Age

## 🎯 Output

* Loan Approved ✅
* Loan Rejected ❌
